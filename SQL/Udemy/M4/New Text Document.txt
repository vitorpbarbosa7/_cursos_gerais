Restrições podem ser aplicadas nos tipos de dados:

NOT NULL -  Não permite nulos 

UNIQUE - Força que todos os valores em uma coluna sejam diferentes

PRIMARY KEY - Uma junção de NOT NULL E UNIQUE (faz todo sentido)

FOREIGN KEY - identifica únicamente uma linha em outra tabela 

CHECK -  Força uma condição específica em uma colunao

DEFAULT - Força um valor padrão quando nenhum valor é passado
